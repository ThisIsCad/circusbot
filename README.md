README
======

Use Node.js 16 or higher.

```
npm install
npm run build; node .\cirque-bot.js
```

Sample `config.json` file:

```
{
    "BOT_TOKEN": "YOUR_SECRET_BOT_TOKEN",
    "BOT_PREFIX": "!",
    "WHITELISTED_CHANNELS": [
        "814616443919532062",
        "730401144773410927"
    ]
}
```
